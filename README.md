<h1> SagaHelper </h1>
<p>SagaHelper is a fan-made tool app designed to enhance your Arknights experience! Whether you're a seasoned Doctor or just starting your journey in Rhodes Island, this app provides essential information and handy tools to help you strategize, optimize, and learn the game to its fullest.</p>
<img src="https://raw.githubusercontent.com/elbriant/elbriant/refs/heads/main/sagahelper_mockup.png" alt="SagaHelper mockup image" style="max-width:500px;width:100%"/>
</center>

## Features
- Operator Database: Quickly access detailed stats, skills, and profiles for all Arknights operators.
- Actual Information: Get server information in your local time
- Custom Themes: use custom themes to personalize your experience

## Downloads
Get the lastest version in [releases](https://github.com/elbriant/sagahelper/releases)
select apk based on your android architecture, if you don't know which one to download you can download app-release.apk (it's bigger but contains all ABIs)
